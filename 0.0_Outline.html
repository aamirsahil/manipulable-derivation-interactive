Derivation: Describing reality using symbols




The process of derivation describes the process of arriving at a symbol-based model of a set of physical phenomena, such as bodies moving in a straight line, rotating bodies, oscillating bodies, etc. This symbol-based model can replicate the behavior exhibited by the physical phenomena and predict it too. This prediction feature makes the symbolic model very useful, as it allows building and exploring many 'what-if' scenarios, without making (costly) changes to the actual system in the world
