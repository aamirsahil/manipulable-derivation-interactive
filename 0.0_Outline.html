Derivation: Building a symbol-based system that can 'act out' the behavior of phenomena in the real world




The process of derivation builds a symbol-based system, which can 'act out' the behavior of a set of physical phenomena, such as bodies moving in a straight line, rotating bodies, oscillating bodies, etc. This system can be thought of as a 'machine', which can mimic, that is act out, behaviors in the real world. When given different values for the symbols, this system can replicate the behavior exhibited by the physical phenomena, using numbers.

Exploring the different states of the symbol-based system then becomes similar to exploring the world. This prediction feature makes the symbol-based system very useful, as it allows building and exploring many 'what-if' scenarios, without making (costly) changes to the actual phenomena in the world
