Elements in the string

   

Describing the ideal reality using small elements



We have now gained some intuition about the behavior of this string. But can we describe this behavior using simpler elements? In order to find that out, we need to introduce an important assumption in our derivation.
We assume that our string is composed of numerous small mass points which are connected to each other by springs. We call this step discretization. The horizontal distance b/w any two mass points is dx, which tends to zero('tends to' roughly means 'is close to').
